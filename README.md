
# SIMPdf

  

Simple pdf editor while preserving structure and format.

  

## TO Run

`cd backend`

  

`pip install -r requirements.txt`

  

Edit **UPLOAD_FOLDER** and **PDF_TO_HTML_FOLDER** variables in `run.py`

`brew install pdf2htmlEX`
  
Run server: 

`python run.py`

  
  

## Run Frontend

  

`cd frontend`

  

`npm install && npm run start`
